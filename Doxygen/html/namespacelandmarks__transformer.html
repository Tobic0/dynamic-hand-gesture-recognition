Documentation for the landmarks' transformer
This script is used for transforming in an appropriate way the landmarks that get generated by MediaPipe.

In particular the coordinates get transformed based on the image dimensions, afterwards all coordinates are transformed to relative position with respect to the wrist keypoint 0 and later on they get normalized so that all the coordinates are between the range [-1, 1],
