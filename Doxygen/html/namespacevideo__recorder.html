Detailed Description

Documentation for video recorder application.


This code is used for creating a menu overlay for permitting the user to choose which hand gesture to record and once choosed, it records no_sequence of times a video for that particular gesture and saves them inside video/video_frames/raw folder

Function to draw on a opencv image the options menu for selecting the hand gesture to record.

  • _image: where to output the menu text
  • op: if 0 load starting menu, if > 0 but lower than the number of actions, load menu for specified action
