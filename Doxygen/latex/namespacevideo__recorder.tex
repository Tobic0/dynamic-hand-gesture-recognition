Documentation for video recorder application. 

~\newline
 This code is used for creating a menu overlay for permitting the user to choose which hand gesture to record and once choosed, it records no\+\_\+sequence of times a video for that particular gesture and saves them inside video/video\+\_\+frames/raw folder
