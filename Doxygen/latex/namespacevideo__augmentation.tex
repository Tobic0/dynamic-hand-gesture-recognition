\doxysubsection{Detailed Description}
Documentation for video augmentation application. 

~\newline
 This code is used for generating new modified videos from an original video (video augmentation) that were recorder with the \mbox{\hyperlink{namespacevideo__recorder}{video\+\_\+recorder}} or with external tools and put inside the PATH\+\_\+\+VIDEO\+\_\+\+RAW folder. ~\newline
 In particular this program generates no\+\_\+augmentation of videos by modifying the brightness and applying random rotation to copies of the original frames collection.
