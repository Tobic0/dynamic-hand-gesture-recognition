\doxysubsection{Detailed Description}
Documentation for video landmark extraction. 

This program is used to extract the coordinates of the landmarks generated by Google\textquotesingle{}s Media\+Pipe framework from the augmented videos generated with \mbox{\hyperlink{namespacevideo__augmentation}{video\+\_\+augmentation}}. The data extracted is then saved in PATH location.
